**JVM**

	- Linking - Verifies bytecode, prepares static variables, resolves symbolic reference
	- Initialization - Runs static initializers and assigns values to static fields.
2. Execution Engine
	- Interpreter
	- JIT (Just In Time)
	- Garbage
